# Flow Editor

An open source flowchart editor with a simple, responsive and minimal web ui.

## Quickstart

1. Clone the repository ```git clone https://github.com/leoolivi/flowchart-editor.git```
2. Open the folder in your favourite editor ```cd flowchart-editor && code .```
3. Install node packages ```npm install```
4. Run the server ```npm run dev```
5. Open it on the browser
6. Try to modify the code

Live demo at: https://flowchart-open-editor.vercel.app/
